# 🚆 Train Railway Management with XML

A simple web-based railway management system built using **Flask** and **XML**. This project demonstrates how XML can be used to store and manage structured data while serving it through a lightweight Python backend.

---

## 📌 Prerequisites

Make sure you have the following installed:

* Python 3.8 or higher
* pip (Python package manager)

---

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/train-rail-way-management-with-xml.git
cd train-rail-way-management-with-xml
```

---

## 🐍 Create Virtual Environment

### ▶️ On Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

### ▶️ On Windows

```bash
python -m venv venv
venv\Scripts\activate
```

After activation, your terminal should show `(venv)`.

---

## 📦 Install Dependencies

Install the required packages:

```bash
pip install flask
```

Or if a `requirements.txt` file exists:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

### On Linux / macOS

```bash
export FLASK_APP=app.py
export FLASK_ENV=development
flask run
```

### On Windows (CMD)

```bash
set FLASK_APP=app.py
set FLASK_ENV=development
flask run
```

### On Windows (PowerShell)

```bash
$env:FLASK_APP="app.py"
$env:FLASK_ENV="development"
flask run
```

---

## 🌐 Access the Application

Once the server is running, open your browser and go to:

```
http://127.0.0.1:5000/
```
